# 🚀 Node.js Express API

Projeto desenvolvido durante o desafio da DIO com o objetivo de criar uma API utilizando Node.js e Express.

---

## 🚀 Tecnologias Utilizadas

- Node.js
- Express
- TypeScript

---

## 📂 Estrutura do Projeto

```bash
src/
 ├── controllers/
 ├── routes/
 ├── middlewares/
 ├── services/
 └── server.ts
```

---

## ⚙️ Como Executar o Projeto

### 1️⃣ Clonar o repositório

```bash
git clone https://github.com/SEU-USUARIO/SEU-REPOSITORIO.git
```

### 2️⃣ Instalar dependências

```bash
npm install
```

### 3️⃣ Executar o projeto

```bash
npm run start:dev
```
